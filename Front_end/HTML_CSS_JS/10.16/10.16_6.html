웹이란? UL

  • 웹은 거미줄을 의미하는데 WWW(World Wide Web)의 약어
  • 인터넷과 웹 브라우저를 통하여 정보를 사용자에게 제공
  • 웹 브라우저를 통하여 원격에 있는 사용자와 상호 간에 소통
  • 웹 관련 직업 : 웹 디자이너, 웹 퍼블리셔, 웹 프로그래머

라면 맛있게 끓이는 법 OL

  1. 종이컵 3컵 분량의 물을 넣어 준다.
  2. 수프를 면보다 먼저 넣고, 물이 끓으면 면을 넣는다.
  3. 면을 젓가락으로 들어 올려주면서 끓인다.
  4. 처음부터 끝까지 센 불을 유지해야 한다.
