<nput size="숫자"> : 입력 필드의 너비 지정






<input maxlength="숫자"> : 최대 입력 문자 수 지정






<input multiple> : 파일 선택 등 입력 필드에서 둘 이상의 값 사용



<input pattern="속성값"> : 정규식 패턴 지정

<input placeholder="설명"> : 입력 형식에 대한 설명 입력 - 숫자 + 하이픈 자리 맞춤

<input required> : 필수 입력 필드로 지정



<select> - dropdown 형태의 목록으로 표시 - <option> 태그로 목록의 요소 생성 - selected 속성으로 기본 선택값 지정
